Após a API ser tivada, em um segundo console R, é criado um json simulando o input de métricas,
conforme o dataset, no qual realiza uma requisição para API, e obtem os resultados, 
printando na tela. Segue a requisição:

library(httr)
library(jsonlite)

# Criar o JSON com os valores das métricas
entrada <- list(
  COUPLING_BETWEEN_OBJECTS = 10,
  DEPTH_INHERITANCE = 2,
  LACK_OF_COHESION_OF_METHODS = 85,
  FAN_IN = 1,
  RESPONSE_FOR_CLASS = 30,
  WEIGHTED_METHODS_PER_CLASS = 25,
  sumLOC_TOTAL = 400
)

# Fazer a requisição POST
resposta <- POST(
  url = "http://localhost:8000/prever",
  body = toJSON(entrada, auto_unbox = TRUE),
  encode = "json"
)

# Ver resultado
content(resposta, as = "parsed", simplifyVector = TRUE)
